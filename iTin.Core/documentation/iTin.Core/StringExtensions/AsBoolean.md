# StringExtensions.AsBoolean method

Tries to convert the value specified in its boolean equivalent value. Default value is false.

Supported values are: "true", "false", "yes", "no", "si", "on", "off", "t", "f", "y", "n", "1", "0".

```csharp
public static bool AsBoolean(this string value)
```

| parameter | description |
| --- | --- |
| value | The value to convert. |

## Return Value
